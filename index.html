Java OpenGL Eclipse Plugin

View on GitHub Download .zip Download .tar.gz

Description

JOGL is a Java binding for the OpenGL API actively maintained by JogAmp.org. This page provide the resources to integrate such binding into Eclipse and RCP applications directly from a repository

Status

The only platform currently supported is Mac OS X

Update Site

Add the following repository to your Eclipse installation

https://cooked.github.io/jogl-eclipse-plugin/update
